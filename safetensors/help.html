safetensors Help

Open a .safetensors file to inspect its metadata and tensor list.

Opening Files

  • Choose File > Open, then select a .safetensors file.
  • Double-click a supported file in Finder to open it in safetensors.
  • Use Quick Look in Finder for a lightweight preview.

Viewing Data

The viewer shows tensor names, shapes, data types, and file metadata in a sortable table.
